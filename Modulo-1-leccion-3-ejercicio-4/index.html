imágenes entre párrafos

Se pueden combinar imágenes y textos utilizando el atributo float en el elemento imagen img. Con esto se consigue que las imágenes queden ubicadas a derecha o izquierda (ver imagen más abajo) y respectivamente los textos a la izquierda o a la derecha de las mismas.

El margen vacío alrededor de la imagen se consigue utilizando el atributo padding (en este ejemplo el margen está fijado en 15 pixeles (Xpx). El texto y la imagen pueden encontrarse dentro de instrucciones como div o formar parte de un párrafo p. También pueden aparecer en forma sucesiva (párrafo, imagen con float, otro párrafo). Las combinaciones son muy versátiles: con esta configuración las imágenes funcionan como si fueran un párrafo más; por eso, los textos e imágenes, podrían encontrarse por ejemplo dentro de un elemento general como div.
