Colorear Links

Título

Colorear enlaces HTML puede ser una gran manera de distinguirlos de su texto regular, o agregar un poco de estilo a su sitio web. Aquí le mostraremos cómo el color Hex enlaces HTML utilizando códigos de color, RGB y HSL valores y nombres de colores HTML.

Nombres de colores HTML son a menudo más legible que sus contrapartes de código Hex, y puede utilizarse en la misma forma. Sustituir el código Hex de color con el nombre de color HTML del ejemplo anterior y voila, el código es muy claro.
